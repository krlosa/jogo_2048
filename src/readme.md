Atividade feita com o intuito de praticar ensinamentos.
2048 é um jogo de raciocínio e estratégia em que o objetivo é combinar blocos numerados para formar o bloco 2048.
Como jogar
Cada movimento desliza todos os blocos na direção escolhida até que encontrem outro bloco ou a borda do tabuleiro .
Quando dois blocos com o mesmo número se encontram, eles se fundem em um bloco com o valor da soma dos dois. Por exemplo, dois blocos “2” se tornam “4”, dois blocos “4” se tornam “8” e assim por diante.
A cada jogada, um novo bloco (2 ou 4) aparece aleatoriamente em um espaço vazio.
O objetivo principal é criar um bloco com o número 2048, embora muitos jogadores continuem para alcançar pontuações ainda maiores.
